Binary Search Algorithm Steps

Step 1: Set Boundaries

Initialize left boundary at index 0 and right boundary at array length - 1.

Step 2: Find Middle

Calculate middle index: mid = (left + right) / 2.

Step 3: Compare with Target

Compare the middle element with the target value.

Step 4: Adjust Search Space

If target is smaller, search left half. If larger, search right half.

Step 5: Repeat or Conclude

Repeat until target is found or search space is exhausted.

Time Complexity:

Best Case: O(1) - element found at middle

Average Case: O(log n)

Worst Case: O(log n)

Note: Array must be sorted!
