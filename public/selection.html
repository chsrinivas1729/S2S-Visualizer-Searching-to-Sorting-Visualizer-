Selection Sort Algorithm Steps

Step 1: Find Minimum Element

Start from the first unsorted position and find the minimum element in the remaining unsorted array.

Step 2: Swap Elements

Swap the found minimum element with the first element of the unsorted portion.

Step 3: Move Boundary

Move the boundary of the unsorted subarray one element to the right.

Step 4: Repeat Process

Repeat the process for the remaining unsorted elements until the entire array is sorted.

Step 5: Complete

The algorithm is complete when no unsorted elements remain.

Time Complexity:

Best Case: O(n²)

Average Case: O(n²)

Worst Case: O(n²)
