Linear Search Algorithm Steps

Step 1: Start from Beginning

Begin searching from the first element of the array (index 0).

Step 2: Compare Current Element

Compare the current element with the target value we're searching for.

Step 3: Check Match

If the current element matches the target, return the index (found!).

Step 4: Move to Next

If no match, move to the next element in the array and repeat the comparison.

Step 5: End Condition

Continue until element is found or end of array is reached (not found).

Time Complexity:

Best Case: O(1) - element found at first position

Average Case: O(n)

Worst Case: O(n) - element at last position or not found
